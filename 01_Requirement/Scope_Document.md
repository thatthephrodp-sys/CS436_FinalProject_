โครงการ: Web Application เครื่องคิดเลข (Calculator Web App)

1. เอกสารขอบเขตโครงการ (Scope Document)
    1.1 วัตถุประสงค์หลัก (Project Objectives)
        เพื่อพัฒนาระบบเครื่องคิดเลขบนเว็บไซต์ที่สามารถใช้งานได้ผ่านเว็บเบราว์เซอร์ทั่วไป โดยไม่ต้องติดตั้งโปรแกรม
        เพื่อรองรับการคำนวณทางคณิตศาสตร์พื้นฐานและการคำนวณทางวิทยาศาสตร์เบื้องต้น
        เพื่ออำนวยความสะดวกให้ผู้ใช้งานสามารถดูประวัติการคำนวณย้อนหลังได้
        เพื่อให้มีหน้าจอการใช้งาน (User Interface) ที่รองรับการแสดงผลทั้งบนคอมพิวเตอร์ (Desktop) และโทรศัพท์มือถือ (Mobile)

    1.2 ขอบเขตของงาน (Project Scope)
        สิ่งที่อยู่ในขอบเขต (IN-SCOPE)
            ฟังก์ชันการคำนวณ:
            การคำนวณพื้นฐาน (บวก, ลบ, คูณ, หาร, เลขทศนิยม)

        ฟังก์ชันการใช้งาน:
            ระบบล้างค่าทั้งหมด (AC) และลบตัวเลขทีละหลัก (Backspace)
            ระบบแสดงประวัติการคำนวณ (History Log) ย้อนหลัง 10 รายการล่าสุด
            รองรับการกดปุ่มผ่านแป้นพิมพ์คอมพิวเตอร์ (Numpad support)

    Non-Functional
        Responsive Design: หน้าจอปรับขนาดตามอุปกรณ์ (Mobile/Tablet/Desktop)
        Theme: รองรับ Dark Mode และ Light Mode

สิ่งที่อยู่นอกขอบเขต (OUT-OF-SCOPE)
    เพื่อป้องกัน Scope Creep ระบบจะไม่รองรับ:
        การพล็อตกราฟ (Graphing features)
        การแปลงค่าเงินหรือหน่วยวัดที่ต้องดึงข้อมูล Real-time (Currency/Unit Converter)
        ระบบสมาชิก (User Login) หรือการบันทึกประวัติข้ามเครื่อง (Cloud Sync)
        การสั่งงานด้วยเสียง (Voice Command)
        การส่งออกข้อมูลเป็นไฟล์ (Export to Excel/PDF)
