รายละเอียดการออกแบบ UX/UI (UX/UI Design Specification)

โครงการ: Web Application เครื่องคิดเลข (Calculator Web App)
อ้างอิงจาก: Mockup ภาพรวม (Screenshot 2025-11-29 210312.png)

1. แนวคิดการออกแบบ (Design Concept)
Style: Minimalist & Flat Design (เน้นความเรียบง่าย แบนราบ และสะอาดตา)
Shape: Geometric (เน้นรูปทรงวงกลมสำหรับปุ่มกด เพื่อความเป็นมิตรและรองรับการกดสัมผัสได้ง่าย)
Platform: Mobile-First Design (ออกแบบโดยเน้นการใช้งานบนมือถือเป็นหลัก แต่รองรับการขยายผลไป Desktop)

2. โครงสร้างหน้าจอ (Layout & Structure)
หน้าจอแบ่งออกเป็น 2 ส่วนหลัก ตามแนวตั้ง (Vertical Layout):

    2.1 ส่วนหัวและหน้าจอแสดงผล (Header & Display Area)
    พื้นที่: ครอบคลุมพื้นที่ด้านบนประมาณ 35-40% ของหน้าจอ
    องค์ประกอบ:
    มุมซ้ายบน: ไอคอนนาฬิกา (History Icon) สำหรับดูประวัติการคำนวณ
    มุมขวาบน: ไอคอนจุดสามจุด (Kebab Menu Icon) สำหรับเมนูเพิ่มเติมหรือการตั้งค่า
    พื้นที่แสดงผล: พื้นที่ว่างขนาดใหญ่สำหรับแสดงตัวเลขผลลัพธ์ (Result) และสมการที่กำลังพิมพ์ (Expression) จัดชิดขวาหรือตรงกลาง (ตามความเหมาะสมของการ implement)

    2.2 แป้นพิมพ์ (Keypad Area)
    พื้นที่: ครอบคลุมพื้นที่ด้านล่างประมาณ 60-65% ของหน้าจอ
    รูปแบบ: Grid Layout ขนาด 4 คอลัมน์ x 5 แถว (4x5)
    ระยะห่าง (Gaps): มีช่องว่างระหว่างปุ่มที่สม่ำเสมอ เพื่อป้องกันการกดผิด (Fat-finger error)
3. ระบบสีและธีม (Color System & Themes)
ระบบมีการออกแบบรองรับ 2 ธีมหลัก คือ Light Theme และ Dark Theme

    3.1 Light Theme (ธีมสว่าง)
    Background (พื้นหลังหลัก): สีขาว (#FFFFFF)
    Primary Button (ปุ่มตัวเลขและเครื่องหมายทั่วไป):
    สีพื้นหลัง: สีม่วงสด (#9C27B0 หรือใกล้เคียง)
    สีตัวอักษร: สีดำ (#000000)
    Accent Button (ปุ่มเท่ากับ =):
    สีพื้นหลัง: สีแดงอมส้ม (#F44336 หรือ #FF5722)
    สีตัวอักษร: สีดำ หรือ สีขาว (แนะนำให้ตรวจสอบ Contrast ratio)
    Icons (ไอคอนส่วนหัว): สีดำ

    3.2 Dark Theme (ธีมมืด)
    Background (พื้นหลังหลัก): สีดำสนิท (#000000)
    Primary Button (ปุ่มตัวเลขและเครื่องหมายทั่วไป):
    สีพื้นหลัง: สีเทาเข้ม (#333333 หรือ #2C2C2C)
    สีตัวอักษร: สีขาว (#FFFFFF)
    Accent Button (ปุ่มเท่ากับ =):
    สีพื้นหลัง: สีส้มสด (#FF9800)
    สีตัวอักษร: สีขาว
    Icons (ไอคอนส่วนหัว): สีขาว

5. การเลือกใช้ตัวอักษร (Typography)
    Font Family: Sans-serif (ไม่มีเชิง) ลักษณะโค้งมน เพื่อให้เข้ากับรูปทรงปุ่มวงกลม (แนะนำ: Roboto, Open Sans, หรือ Poppins)
    Font Weight: Bold (ตัวหนา) สำหรับตัวเลขบนปุ่ม เพื่อให้อ่านง่ายและชัดเจนที่สุด
    Text Size: ขนาดตัวอักษรบนปุ่มควรมีขนาดใหญ่ (เช่น 24px - 32px) เพื่อให้อ่านง่ายบนมือถือ

6. สถานะการใช้งาน (Interactive States - Recommendation)
    Normal State: สีปกติตามที่ระบุในข้อ 3
    Active/Pressed State: เมื่อผู้ใช้กดปุ่ม
    Light Theme: ปุ่มควรมีสีเข้มขึ้นเล็กน้อย หรือมีความสว่างลดลง
    Dark Theme: ปุ่มควรสว่างขึ้นเล็กน้อย (Lighten) เพื่อแสดง Feedback ว่ามีการกด
